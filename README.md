## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/novelrepo.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the app:**
   ```bash
   npm start
   ```
